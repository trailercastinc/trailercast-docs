| Key | Where it works | What it does |
|---|---|---|
| `[` | On any video with a chapter strip | Jump to previous chapter |
| `]` | On any video with a chapter strip | Jump to next chapter |
| `Esc` | Any modal | Close the modal |
| `Cmd/Ctrl + Enter` | Ask Winter / Comments composer | Send message |
| `@` | Comments composer | Open mention picker |
| `Cmd/Ctrl + F` | Transcript view | Search within transcript |

<Note>
  Keyboard shortcuts only fire when focus is OUTSIDE any text input, they won't hijack your typing in the Comments composer or Ask Winter chat.
</Note>
