<Update label="2026-05-08" description="Pricing + positioning refresh">
  ### Pricing reset
  - **Starter** now $100/mo (down from $200), still 1 seat, 10 demos/mo
  - **Pro** moves to per-user pricing: $70/user/mo with a 3-user minimum (so $210/mo entry, replacing the old flat $500/mo). Add seats from Settings → Billing as you grow.
  - Annual is now expressed as **2 months free** (effectively ~17% off) instead of "saves ~20%"

  ### Positioning refresh
  - Brand framing now leads with "meeting recording tool with AI analysis built in" before introducing trailers, so prospects coming from a Gong / Avoma evaluation immediately understand the call-recording side of the product
  - Trailer length expectations corrected: trailers are **10–15 minutes**, not 60–180 seconds. Long enough to carry context, short enough that prospects finish them.
  - Brand Settings significantly expanded: now covers intro slides, outro/end cards, agenda slides (auto-populated with chapter list), transition slides, lower-third graphics, background music, and trailer templates. Set up once during onboarding, every trailer inherits.
</Update>

<Update label="2026-05-08" description="Avoma-style call detail + AI personalization injection">
  ### Trailer review surface, major rebuild
  - **Color-coded chapter strip** under every video player (call detail, demo trailer, share viewer)
  - **Active-segment highlight** as the video plays
  - **Keyboard shortcuts** `[` / `]` to jump between chapters
  - **Avoma-style 2-column call detail page** with tabbed right rail (Summary / Topics / Transcript / Comments / Ask Winter)
  - **Demo trailer 2-column layout** with 4-tab right rail (Engagement / Highlights / Comments / Ask Winter)

  ### AI personalization Level 1
  - The AI Personalization form's data is now **read by every AI inference point**, moment detection, summaries, deal briefs, narration, Ask Winter
  - Previously only moment detection was personalized; now every output adapts to your stated context

  ### Comments + Ask Winter
  - **Comments tab badge** with count on call + demo detail
  - **Ask Winter conversations persist** across tab switches and refreshes (sessionStorage, per-recording)
  - **Global Ask Winter pill** auto-attaches current page context, ask about "this call" without spelling out which one

  ### Bug fixes
  - "Watch full segment" pill now jumps to the START of the clip (was incorrectly jumping to the end since 596c8d0b)
  - Meeting summaries now generate correctly (had a function-name typo silently failing for an unknown duration)
  - Talk-time timeline labels no longer collide on narrow screens
  - Detail pages no longer feel "containerized", fully edge-to-edge
</Update>

<Update label="2026-04-28" description="Granular permissions + private recordings">
  - **3-role permissions system**: Global Admin (immutable, 1/co), Admin (defaults all-on), Member (curated)
  - **Per-permission overrides** with lock-by-Global-Admin mechanic
  - **Transfer Global Admin** flow
  - **Private recordings**. Global Admin can mark any recording private; invisible to other team members
  - **Don't record** action on upcoming bots
  - **Reclassify** action via kebab menu on every recording
</Update>

<Update label="2026-04-25" description="Edge-to-edge layout + universal search">
  - Layout migration: dropped the boxed-in card chrome, max-width bumped to 1600px
  - **New TopBar** with universal search across demos, calls, Buyer Spaces, and team
  - Custom logo upload (replaces TrailerCast brand on the top bar)
  - Mobile: search collapses to icon, opens overlay
  - **Default Upcoming view is now List** (not Calendar)
</Update>

<Update label="2026-04-20" description="Calendar polling + meeting bots">
